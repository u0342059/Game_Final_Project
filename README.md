## 主題:
### 跑酷遊戲-消滅蛋頭博士

## 內容簡介:
- 音速小子索尼克要消滅死對頭蛋頭博士
- 蛋頭博士會坐著飛行器向索尼克投射飛彈，還會派出機器蟲進行攻擊。
- 索尼克必須跳躍障礙物及發射圓環擊敗敵人才能得分

## 遊戲操作
- 一進到遊戲，會先到開始畫面，點下”START”即可開始
- 點選螢幕右半邊可發射圓環
- 點選螢幕左半邊角色人物可跳躍
- 遇到敵人則可以使用圓環消滅
- 敵人分為Boss及小怪
- 碰到障礙物或是敵人時則會結束遊戲
- 跳出GameOver視窗，選取“YES”會再一次進行遊戲，“NO”則會退出遊戲
